Welcome to the Priority User Guide

Priority Home Page

Your starting point. Search for data, access frequently used forms, change settings, track changes, chat with colleagues, and view data visualizations.

Click any element below to explore:

🏠 Home Page Button
Return to home from anywhere
Click this button at any time to return to the Priority Home Page from any form, report, or program.
⭐ Favorites Menu
Your personalized menu
Create a personal menu tailored to your needs. Add frequently used forms, programs, and reports. Organize with folders. Access via the star icon at top of screen. Add entities from any form by clicking the star in the form header.
🔎 Entity Search
Find forms, programs, reports
Click magnifying glass on left of Menu bar. Type a term and press Enter. From results: Run the entity, view Help, or More to copy link, view path, add to Home Page. Case-sensitive unless "Ignore Case" is checked.
📂 Priority Menu
All forms, programs, reports
Top of screen. Each item = a module (Purchasing, Production, etc.). Entity types: Menu (submenu), Form (records), Report, Program, Printout. Navigate: Enter/→ activate, ↑↓ move, ← close. Right-click for Help, Copy Path, Add to Home Page.
🔍 Search Bar
Search documents and data
Located under the Priority Menu. Type customer names, order numbers, etc. Results appear as you type. Click search icon for full results sorted by type.

Special characters:
"..." — Exact phrase
+ — Must appear
- — Exclude (no space)
$ — Exact word only
🧩 Widgets
Shortcuts to entities
Shortcuts to frequently used entities. Add: click + icon or right-click entity → "Add to Home Page". Remove: hover → More Options → Delete. Rearrange by dragging. Badge = open documents assigned to you. Hover for search and new record icons.
💬 Priority Talk
Notifications and chat
Track document changes and chat with colleagues. Click the bell icon when alerts are detected to view the Alerts window.
📈 Portlets
Visual data graphs
Display company data (sales orders, P&L, etc.) as graphs. Click chart icon to add. Options: choose dates, open in new window, delete. Requires BI module. Run "Prep for Inv. Analysis (BI)" daily.
📦 Packages
Organize workspace by subject
Each package = forms + fields + programs for a topic. One click enables/disables a package. Hide irrelevant modules to declutter your workspace.
🕐 Recent & Updates
Recent items and document changes
Recent Items: Automatically listed on Home Page.
Latest Updates: Documents opened/updated in past 30 days. System constants LATESTUPDATES and LOGUPDATE control what appears.
📱 Active Screens
Open forms indicator
Shows count of open forms. Click to see list. Click form to switch, hover + x to close, or "Close All".
⚙ Options Menu
Settings, language, Help
Change company, password, language, work area design, internal mail, and Help. Additional functions available when forms are open.
❓ Help System
Built-in explanations
Help icon (upper-right): "Get to Know" document, Priority Xpert Knowledge Base (videos, FAQs). Right-click entity → Help. F1 for field help. Online Help has SOPs, FAQs, videos.
📈 Analytics
Build personalized reports
Build, view, run, and edit personalized reports. Access reports you built or that were shared with you.

Entering Priority

Open Priority ▶

When you open Priority, the Home Page is displayed. You can begin working immediately. The system supports multiple companies — switch via the Options Menu.

Help Options on Entry ▶

Help Button (upper-right): "Get to Know: Buttons, Icons, and Keys" document, Priority Xpert Knowledge Base with videos and FAQs.

Entity Help (F1): Right-click any entity in the Priority menu → Help. Choose "Online Help" for SOPs, FAQs, and video clips.

Updating Records ▶

Retrieve the record (via search or scrolling), click the field to change, modify the value. Changes save when you move to the next field. Focus View is recommended for editing — all fields visible at once.

ℹ️
F5 — Choose subform and jump to it. Shift+F5 for the same in search mode.
Text Forms & Attachments ▶

Text Forms: Add free-text remarks that can appear on printed documents.

Attachments: Ctrl+I to attach files. Click Settings to choose display info (name, date, size).

Copy/Paste: Ctrl+C/V standard. Ctrl+H copies entire column. F10 copies from above. Ctrl+F9 saves to buffer, F9 pastes from buffer.

Form Actions & Sharing ▶

Ctrl+F5 opens the Actions list: run programs, generate printouts, send emails, open related forms.

Sharing: Ctrl+S — mail with record link. Ctrl+L/Y — copy link. Ctrl+O — send link to participants.

Tracking Changes ▶

Click the bell icon for alerts. The History window shows a log of changes. Use Settings to customize what types of changes appear.

BI Reports & Drill-Down ▶

Analyze data from multiple perspectives. Access from Executive Reports module, analysis menus, form fields with BI icon, or Options Menu → Run → Analysis Reports (BI).

One-dimensional: Each level appears alone. Drill down through cross-sections. Back arrow to return.

Multi-dimensional: View different cross-sections in a single screen within a hierarchical tree. Expand/collapse nodes.

Filter with patterns/wildcards, set periods (absolute or date ranges), choose Level of Detail (monthly, quarterly), save period templates.

ℹ️
Run "Prepare All for Analysis (BI)" daily for current data (usually via Tabula Task Scheduler).
Parameter Input — Complete Guide ▶

Use mouse, Tab, or arrow keys between fields. Don't press Enter until all input is complete.

MethodHow
Exact valueSelect from drop-down or type
Wildcard (*)Default — all records. E.g., F* = starting with "F"
OR (|)Duplicates field. Enter values in each — either matches
BETWEEN (-)First value left, last right — range only
AND NOT (!)Include left, exclude right. E.g., 2* except 21*
OperatorsClick left arrow: =, <, <=, >, >=, <>
💡
Date tip: F6/Space for relative dates (yesterday, next month). Shift+F6 for pop-up calendar.
Dashboards ▶

Multiple mini-windows (web parts): reports, tables, graphs, gauges, pies. Click design icon to rearrange/add/hide web parts. Add external web sites. Gauges use red/yellow/green for targets — configure in "Definition of Gauge Scales" form. Click GO to apply changes.

Personalize Your Priority

Extensive customization: see only relevant entities, rearrange layouts, design forms to match your workflow.

Automated Recommendations ▶

Priority suggests personalization based on usage. The Personalization Center icon appears when recommendations are available. "Maybe later" items can be found under the same icon.

Design by Packages — Step by Step ▶
Customize Menus ▶

Design Tool options: Move items up/down, Hide items, Add User-Defined Reports (admin). Right-click menu items for design options. Hidden menus show a special icon.

Favorites Menu ▶

Click star icon in any form to add/remove from Favorites. Organize: click star at top → right-click entity → "Organize Favorites", or Options Menu → Design → Organize Favorites.

Customize Forms — Design Tool ▶

Three modes:

1. Fields & Tabs: Rearrange, hide/display, space, merge, freeze fields.

2. Subforms: Reorder tabs, hide unused subforms.

3. Actions: Choose which appear as smart buttons vs. in Actions list.

💡
Revert: Design Tool has a reset option to restore original layout. Define record colors via right-click → Record Color. Adjust column widths by dragging.
Customize Reports ▶

Use Design Report tool to change field order, add/remove fields, modify widths, select print formats. Most documents have predefined formats you can modify.

Automations Hub

Record macros, define business rules, and set up autofill rules to automate repetitive tasks.

Recording Macros — Step by Step ▶

Three main macro types:

TypeUse Case
Single-form, non-repeatingSame data updates in different documents
Single-form, repeatingSame action down multiple lines
Multi-formNavigate between forms and perform actions
Business Rules — Step by Step ▶

Rule types: Warning, Error, Email, Text message, Push notification, User verification.

Autofill Rules ▶

Automatically set field values when conditions are met. Example: set salesperson when customer is selected. Use function expressions for calculated values (e.g., discount based on quantity).

ℹ️
Use AND (all conditions) or OR (any condition) in the When section. Valid For determines whose actions trigger the rule.
Managing Rules ▶

Search by keyword. Report via More → Report. Reorder with arrows. Edit by clicking. Delete to remove.

Advanced: Templates

Create Word documents, spreadsheets, labels, and letters from system data.

Word Templates — Step by Step ▶
Excel Spreadsheet Templates ▶

Generate spreadsheets from system data. Create templates from forms or reports using the Design Template dialog. Map fields to columns, save, then run to generate Excel files with live data.

To edit: locate template, click edit icon, download, modify in Excel, save back.

Label Generator ▶

Create labels from form data automatically. Generate unlimited labels from individual records. Choose between preparing for current record or all displayed records. Edit in Work Area or print immediately.

ℹ️
Requires a label printer and label printing software.
Letter Generator ▶

Produce pre-formatted letters using data from Priority forms. Include attachments. Print or send via email. Use "Letter to Customer" program (CRM → Customers → Customer Reports). Also available for vendors and contacts.

BPM Flow Charts

Define document workflows — statuses, transition paths, and rules at each stage.

BPM Basics ▶

Each flow chart has a default initial status (usually "Draft") and defines: which statuses exist, which transitions are allowed, what rules trigger at each change, and who can change statuses. Tool Tray in lower-left has shortcuts.

Creating Status Paths — Step by Step ▶
Rules for Statuses ▶

Set up rules for status events:

• Automatic email/text/push when a status is reached
• Escalation — re-assign if document stays too long in a status
• Open a task on status change
• User verification — require confirmation
• Allow/block changes based on conditions

Same AND/OR logic and Valid For settings as Automations Hub business rules.

BPM Error Reports ▶

If an escalation rule triggers a change that form logic doesn't permit (e.g., setting "Final" on an un-itemized shipment), the system generates an error report explaining what happened and why.
